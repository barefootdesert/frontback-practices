# Практическое занятие №1 — CSS-препроцессоры

**Дисциплина:** Фронтенд и бэкенд разработка  
**Семестр:** 4, 2025/2026 уч. год

## Что реализовано

- Переменные (`$primary`, `$accent`, `$radius`, `$theme`)
- Вложенность селекторов (`&__title`, `&__button`)
- Миксины (`button()`, `card-hover()`)
- Условные конструкции (`@if $theme == dark`)
- Hover-эффекты и современный дизайн

## Как запустить

```bash
# 1. Установка SASS (один раз)
npm install -g sass

# 2. Компиляция SCSS в CSS
sass styles.scss styles.css

# 3. Автокомпиляция (рекомендуется)
sass --watch styles.scss styles.css
